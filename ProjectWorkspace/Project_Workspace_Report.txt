Created a ProjectWorkspace directory with docs, src, and logs subfolders. Before modification, project_notes.txt had permissions -rw-r-r--(644). After running chmod 750, it changed to -rwxr-x--(750), restricting "others" from any access, protecting sensitive project data from unauthorized users. Ownership is set to my user account. The system umask is 0022, meaning new files default to 644 and new directions to 755 unless explicitly changed.
